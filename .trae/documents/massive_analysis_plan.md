# PPS vs Y 大规模分析计划

## 分析目标

全面、系统地理解 PPS 和 Y 的对应关系：

1. 明确所有展开规则（后继、极限、其他）
2. 找出什么时候 pps(...) = y(...)
3. 分析极限对应关系
4. 通过大量测试验证规律

## 分析原则（来自 MEMORY.md）

1. 从小到大，从简单到复杂
2. 数学计算原则优先
3. 谨慎推断，不轻易下结论
4. 追求详细，深入分析

## 分析阶段

### 第一阶段：基础对应关系梳理

* [ ] 回顾已有的所有测试数据

* [ ] 从数学上计算 pps(...) 和 y(...) 的值

* [ ] 尝试找出 pps(...) = y(...) 的条件

* [ ] 分析极限展开的对应规则

### 第二阶段：系统性测试

* [ ] 严格递增序列：长度 2, 3, 4, 5...

* [ ] 末尾重复序列：重复 2 次、3 次...

* [ ] 中间重复序列：非末尾的重复

* [ ] 递减序列（如果合法）

* [ ] 含0的 PPS 序列 vs 含1的 Y 序列

* [ ] 交替模式：`0 1 0 1 0 1...` vs `1 2 1 2 1 2...`

### 第三阶段：规则性分析

* [ ] 分析后继展开的规律

* [ ] 分析极限展开的规律

* [ ] 分析特殊元素（0 或 1）的影响

* [ ] 分析序列长度的影响

* [ ] 分析重复次数的影响

### 第四阶段：极限对应关系

* [ ] 对比 pps(A) 和 y(A+1) 的数学定义

* [ ] 找出什么时候 pps(A) = y(A+1)

* [ ] 分析极限展开中的对应关系

* [ ] 验证 sup 的对应关系

### 第五阶段：深入与总结

* [ ] 整理所有发现

* [ ] 总结规律

* [ ] 记录例外情况

* [ ] 形成完整的对比文档

## 实施原则

> 从简单到复杂，逐步推进
