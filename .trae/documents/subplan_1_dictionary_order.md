# 小计划1：基础验证（字典序）

## 目标
确认字典序比较方式，分析哪些序列在字典序下有意义。

## 实施步骤

### 步骤1：理解字典序
从左往右比较，找出序列的大小关系。

### 步骤2：分析已知序列的字典序位置

| PPS序列 | Y对应序列 | 字典序位置 |
| :--- | :--- | :--- |
| `0` | `1` | 最小 |
| `0 1` | `1 2` | 小 |
| `0 1 2` | `1 2 3` | 中 |
| `0 1 2 3` | `1 2 3 4` | 大 |
| `0 1 2 2` | `1 2 3 3` | 中（但展开行为不同） |

### 步骤3：测试边界情况
- 测试 `0 1 1`（字典序 < `0 1 2`）
- 测试 `0 1 3`（字典序 > `0 1 2`）
- 测试 `0 1 2 0`（更长但前缀匹配）
