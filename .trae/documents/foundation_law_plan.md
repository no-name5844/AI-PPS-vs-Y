# PPS vs Y 基础规律计划

## 核心原则
- 从最小的开始：pps(0) 和 y(1)
- 只有一种延伸方式：往后要么+1，要么重复最后一个元素
- 通过展开来看极限关系
- 每一步都要支撑下一个结论

## 分析步骤

### 第1步：基础序列验证
- 验证 pps(0) 和 y(1) 的基础行为
- 验证 pps(0, 1) 和 y(1, 2)
- 验证 pps(0, 1, 2) 和 y(1, 2, 3)
- 建立最基础的对应关系

### 第2步：严格递增延伸
- pps(0, 1, 2, 3) vs y(1, 2, 3, 4)
- pps(0, 1, 2, 3, 4) vs y(1, 2, 3, 4, 5)
- 观察极限展开的模式

### 第3步：末尾重复延伸
- pps(0, 1, 2, 2) vs y(1, 2, 3, 3)
- pps(0, 1, 2, 2, 2) vs y(1, 2, 3, 3, 3)
- 观察重复次数的影响

### 第4步：更复杂的延伸（从现有序列继续延伸）
- 从 pps(0, 1, 2, 2) 继续：要么+1得到 0,1,2,3，要么重复得到 0,1,2,2,2
- 从 y(1, 2, 3, 3) 继续：要么+1得到 1,2,3,4，要么重复得到 1,2,3,3,3
- 分析这些分支的极限关系

### 第5步：总结规律
- 总结什么时候两个序列的极限有对应关系
- 记录展开模式的变化
- 找出规则

> 不轻易下结论，除非有足够证据
> 按照"往后+1或重复最后一个"的唯一方式延伸
