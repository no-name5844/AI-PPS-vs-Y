## PPS 到 Y 的转换规则（2026-05-04 更新）

### 正式规则
1. **基础**：`pps_to_y(0) = y(1)`
2. **后继展开**：`pps_to_y(A, 0) = y(pps_to_y(A), 1)`
3. **极限展开**：`pps_to_y(A) = y(B)`，满足区间关系：
   `pps(A[n]) < pps(y_to_pps(B[m])) < pps(A[n+1])`

### 说明
- 规则3：PPS序列A的展开与Y序列B的展开是**交错区间**关系
- 只在极限类型序列上使用规则3
- 断言失败的序列不适用

## 观察与发现

### PPS 系统
- 简单序列 `0 1` 展开为无限重复的 0
- 序列 `0 1 2` 展开为 0 后跟无限重复的 1
- 序列 `0 1 2 2` 展开为交替模式：0,1,2,1,4,1,6,1,8,...
- 序列 `0 1 0` 提前终止
- 序列 `0 1 0 1` 产生复杂的交替模式
- 序列 `0 1 0 0 3` **快速收敛到0**
- 序列 `0 1 2 3` 展开为 0,1,2 后跟无限重复的 2

### Y 系统
- 简单序列 `1 2` 展开为无限重复的 1
- 序列 `1 2 3` 展开为 1,2 后跟无限重复的 2
- 序列 `1 2 3 3` 展开为交替模式：1,2,3,2,3,2,3,...
- 序列 `1 2 1 1 4` **指数增长**：1,2,1,1,3,9,27,81,243,...

### PPS vs Y 关键差异
| 特性 | PPS | Y |
| :--- | :--- | :--- |
| 起始元素 | 0 | 1 |
| 空序列行为 | 无输出 | 断言失败 |
| 正确对应 | `0 1 0 0 3` vs `1 2 1 2 3` | 都收敛，展开行为相似 |
| 错误对应 | `0 1 0 0 3` vs `1 2 1 1 4` | 收敛 vs 指数增长，非对应 |

## 字典序分析（2026-05-04）

### 边界情况测试

| PPS序列 | PPS展开 | Y对应序列 | Y展开 | 字典序关系 |
| :--- | :--- | :--- | :--- | :--- |
| `0 1 1` | `0,1,0,3,0,5,...` | `1 2 2` | `1,2,1,2,1,2,...` | `0 1 1` < `0 1 2` |
| `0 1 3` | 错误 | `1 2 4` | `1,2,3,4,5,...` | `0 1 2` < `0 1 3`（但PPS出错） |
| `0 1 2 0` | `0,1,2,` | `1 2 3 1` | 断言失败 | 前缀匹配，终止方式不同 |

### 重要发现
- **PPS 和 Y 的字典序规则不同**：PPS `0 1 3` 出错，但 Y `1 2 4` 正常
- **字典序不是唯一对应标准**：有些序列在一个系统有效，在另一个系统无效
- **提前终止 vs 断言失败**：`0 1 2 0` 提前终止，但对应的 `1 2 3 1` 断言失败

## 小计划4：严格递增序列分析（2026-05-04）

### 核心观察
对于严格递增序列，PPS 和 Y 表现出**完全对应的模式**：

| PPS序列 | PPS展开 | Y序列 | Y展开 | 对应模式 |
|---------|---------|-------|--------|---------|
| `0` | 无输出 | `1` | 断言失败 | 基础对应 |
| `0 1` | `0,0,0,0,...` | `1 2` | `1,1,1,1,...` | 前缀 = 序列，剩余重复最后一个元素-1 |
| `0 1 2` | `0,1,1,1,1,...` | `1 2 3` | `1,2,2,2,2,...` | 前缀 = 序列，剩余重复最后一个元素-1 |
| `0 1 2 3` | `0,1,2,2,2,2,...` | `1 2 3 4` | `1,2,3,3,3,3,...` | 前缀 = 序列，剩余重复最后一个元素-1 |
| `0 1 2 3 4` | `0,1,2,3,3,3,3,...` | `1 2 3 4 5` | `1,2,3,4,4,4,4,...` | 前缀 = 序列，剩余重复最后一个元素-1 |

### 规律总结
1. **对应关系公式**：对于严格递增序列 `A = [0, 1, 2, ..., n]`，对应的 Y 序列是 `[1, 2, 3, ..., n+1]`
2. **展开模式公式**：
   - PPS展开：前 n+1 个元素 = `A`，剩余无限重复 `n-1`（最后一个元素-1）
   - Y展开：前 n+1 个元素 = `Y(A)`，剩余无限重复 `n`（最后一个元素-1）
3. **极限关系**：两者的 sup（上确界）相等

## 小计划5：末尾重复序列分析（2026-05-04）

### 核心观察
对于末尾重复的序列，PPS 和 Y 都表现出**交替模式**，但细节不同：

| PPS序列 | PPS展开 | Y序列 | Y展开 | 对比 |
|---------|---------|-------|--------|------|
| `0 1 2 2` | `0,1,2,1,4,1,6,1,8,1,...` | `1 2 3 3` | `1,2,3,2,3,2,3,2,3,...` | 都有交替模式，但 PPS 是线性增长，Y 是循环 |

### 模式分析
- **PPS**：交替模式是 `[..., n-1, n+1, n-1, n+3, n-1, ...]`，线性增长
- **Y**：交替模式是 `[..., n-1, n, n-1, n, ...]`，固定值循环
- **共同点**：都表现为「前一个元素」与「后续元素」交替出现
